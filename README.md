## Getting Started

First, run the development server:

```bash
npm run dev
```
Then start sanity

```bash
cd rare
sanity start
```

Open [http://localhost:3000](http://localhost:3000) to view frontend.
Open [http://localhost:3333](http://localhost:3333) to view sanity.
